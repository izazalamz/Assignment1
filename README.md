3. Explain the difference between any, unknown, and never types in TypeScript.
-> টাইপ any হলে সেই ভ্যারিয়েবল নিয়ে যা খুশি তাই (যেকোনো টাইপের ভ্যালু ব্যবহার)করার স্বাধীনতা থাকে কারণ কম্পাইলার টাইপ চেক করে না any টাইপ হইলে। unknown টাইপের ক্ষেত্রেও যেকোনো টাইপের ভ্যালু ব্যবহার করা যায় কিন্তু ব্যবহার করার আগে চেক করে নিতে হয় যে কোন টাইপ এইটা। never টাইপ ব্যবহার হয় যখন অসম্ভব কোনো অবস্থা সৃষ্টি হয় যেমন যেই ফাংশন কোনোদিনও শেষ হবে না অর্থাৎ কোনো শেষ নাই এমন জায়গায় never ব্যবহার করা হয়
